# MRL-Projeto-Frontend
Projeto da faculdade - Terceiro Semestre

## Descrição
Projeto frontend para apresentação do MRL. Os arquivos principais estão em `src/` (páginas HTML e scripts em `src/JS/`).

## Pré-requisitos
- Node.js (>= 12) e npm
- (Opcional) Firebase CLI: `npm install -g firebase-tools`

## Instalação
1. Abra o terminal na raiz do projeto.
2. Instale dependências:
```sh
npm install
```

Rodando localmente
Opções rápidas para servir a pasta src/:

Usando um servidor estático via npx:
npx serve src
# ou
npx http-server src -o
